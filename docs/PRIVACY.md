# 개인정보처리방침 — 우리의 하루

시행일: 2026-07-01

'우리의 하루'(이하 "앱")는 커플이 함께 기록을 남기는 개인용 서비스입니다. 아래와 같이
개인정보를 처리합니다.

## 1. 수집·처리 항목
- 계정: 이메일 주소, 비밀번호(해시로 저장, 원문 미보관)
- 서비스 데이터: 사귄 날, 기념일, 일정, 사진, 일기(데코)·기분·질문 답변, 쿡찌르기 메시지
- 푸시: 브라우저/기기 푸시 구독 정보(알림 발송용)
- 진단 로그: 앱 오류·푸시 상태 로그(문제 해결용)

## 2. 이용 목적
- 커플 간 기록 공유, 기념일 알림, 서비스 제공·오류 대응.

## 3. 저장·처리 위탁
- 데이터는 **Supabase**(인증·데이터베이스·스토리지·실시간·함수) 인프라에 저장·처리됩니다.
- 네이티브 푸시 사용 시 **Google Firebase Cloud Messaging / Apple APNs** 를 통해 알림이
  전달됩니다.

## 4. 보관 및 파기
- 계정/데이터는 이용 중 보관되며, 사용자가 삭제하거나 계정 탈퇴 시 관련 데이터가 삭제됩니다.
- 만료된 푸시 구독은 자동 정리됩니다.

## 5. 제3자 제공·판매
- 개인정보를 제3자에게 판매하지 않습니다. 위 인프라(Supabase/Google/Apple) 외 공유하지 않습니다.

## 6. 이용자 권리
- 데이터 조회·수정·삭제, 계정 탈퇴를 요청할 수 있습니다(앱 설정 또는 아래 연락처).

## 7. 보안
- 전송 구간 암호화(HTTPS), 행 수준 보안(RLS)으로 '내 커플' 데이터만 접근 가능하도록 제한합니다.
